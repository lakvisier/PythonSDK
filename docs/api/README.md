# API Specifications

This directory contains API specification files for reference and validation.

## Files

- **openapi.json** - OpenAPI 3.0 specification for the Visier Platform API
  - Used for understanding API schemas and validating request/response structures
  - Reference for aggregate query payload construction
  - Source of truth for `AggregationQueryExecutionDTO`, `QueryAxisDTO`, `QueryFilterDTO`, etc.

## Usage

This specification is primarily used as a reference when:
- Constructing aggregate query payloads
- Understanding available API endpoints and schemas
- Validating request/response structures
- Discovering available options and parameters
